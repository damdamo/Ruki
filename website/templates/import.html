Here you can import your own data and visualize it after (here)


How to import your own results ?

Firstable, you need to export data if you want to use it. For this, go here to export it.

Now you can use your own methods on this data. If you want to import it in the knowledge base, you have two formats:
CARE: When you import your file, a process is starting to create the visualization for your own results.
So it may take a few minutes, Wait until you get the message: "The file has been successfully added to the database"

        {
