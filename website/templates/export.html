Export data

Here you can export data based on articles on: vgibox.eu

The file you can download contain a json file in this format:

      
        {
         "unique_id": {
          "keywords": "A list of keywords from the article",
          "title": "Title of the article",
          "abstract": "Abstract of the article",
          "id": "ID of the article"
         },
         ...
       }

You can use all of this informations to try to create your own categorization based on this data. Think to look at import page if you want to import it and visualize it.
