Welcome to RUKI !

What's the goal of RUKI ?
It's very simple, i want to offer an easy way to visualize data from: vgibox.eu

Nowadays we have many ways to visualize data, but too many are just printing text. If we can improve this simple method (textual research) and try to think different, we propose something different !
You can find three main part in this website:
Visualization, import and export.
If you want to see result, just go visualization !
If you want to test your method on the same dataset, export it and import it (Explication are given in the import page)
